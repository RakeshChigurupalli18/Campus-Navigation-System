# 🧭 Campus Navigation System

A smart web-based application that helps students navigate their college campus using interactive maps and voice assistance.

---

## 🚀 Features
- 📍 Interactive campus map
- 🔍 Search for locations (labs, classrooms, offices)
- 🎤 Voice-based navigation support
- 🗺️ Easy route guidance inside campus

---

## 🛠️ Tech Stack
- Java (Servlets)
- JSP / HTML / CSS
- Maven
- Leaflet.js

---

## 📂 Project Structure
- src/main/java → Backend logic
- src/main/webapp → Frontend (JSP, HTML)
- pom.xml → Maven configuration

---

## ▶️ How to Run
1. Import project into Eclipse (Maven)
2. Configure Apache Tomcat server
3. Run the project
4. Open: http://localhost:8080/Campus
