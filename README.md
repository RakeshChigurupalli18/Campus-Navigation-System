# 🧭 Campus Navigation System

A smart web-based application that helps students navigate their college campus using interactive maps and voice assistance. The system provides real-time navigation with an intuitive interface.

---

## 🌟 Project Highlights

* Real-time campus navigation system
* Voice-enabled location search
* Interactive map using Leaflet.js
* User-friendly interface

---

## 🚀 Features

* 📍 Interactive campus map
* 🔍 Search for locations (labs, classrooms, offices)
* 🎤 Voice-based navigation support
* 🗺️ Easy route guidance

---

## 🛠️ Tech Stack

* Java (Servlets)
* JSP / HTML / CSS
* Maven
* Leaflet.js

---

## 📂 Project Structure

* `src/main/java` → Backend logic
* `src/main/webapp` → Frontend
* `pom.xml` → Maven configuration

---

## ▶️ How to Run

1. Import project into Eclipse (Maven)
2. Configure Apache Tomcat
3. Run the project
4. Open: http://localhost:8080/Campus
